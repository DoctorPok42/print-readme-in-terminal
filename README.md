# Print_Readme

## Description

This is a simple script to print the README.md file in the terminal.

## Usage

```bash
$ ./print_readme
```

## Type support

| Support | Flags |
| -- | - |
| ✅ | # |
| ✅ | - |
| ✅ | ` |
| ❌ | --- |
| ❌ | ``` |
| ❌ | link |
| ❌ | image |
| ❌ | list |
| ❌ | table |
| ❌ | code block |
